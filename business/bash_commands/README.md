# Bash and Linux basic commands

- Where am i? 

``` pwd ```

- How to create a Dir:

``` mkdir name_of_directory ```

- How to create a file 

``` touch name_of_file```

- how to change dir:

``` cd dir_name ```
- how to come our of Dir:

 ``` cd .. ```

- Who am i? 

``` uname ```

- List Dir and all Dir: 

``` ls ``` or ``` ls -a ```
